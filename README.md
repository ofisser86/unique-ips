# Find all unique IP addresses

## For python scripts

### Print only IP`s

Use `./unique_ips.py <some_log_file> <number_of_last_lines>`

### Print whole line

Use `./unique_lines.py <some_log_file> <number_of_last_lines>`

## For bash script

Use `./unique.sh <some_log_file> <number_of_last_lines>`
